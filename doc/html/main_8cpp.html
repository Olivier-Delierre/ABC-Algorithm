DESCRIPTION

This program use the Artificial Bee Colony algorithm in order to obtain the x value for certain function in order to obtain the minimum of them.

At the moment, you can calculate the minimum of 6 functions, which are mentioned in the Benchmark's header file. Here's the list :

  • Ackley
  • Rastrigin
  • Rosenbrock
  • Schaffer
  • Schwefel
  • Weierstrass
